IMPORTANT COMMANDS:
===================

Install Dependencies:
  cd server && npm install
  cd ../client && npm install

Seed Database:
  cd server
  node seedData.js

Start Backend:
  cd server
  npm start
  (Runs on http://localhost:5000)

Start Frontend:
  cd client
  npm start
  (Runs on http://localhost:3000)
